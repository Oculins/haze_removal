# single image haze removal
## Introduction
This program implement single image dehazing using dark channel prior. 

## Requirements
- OpenCV
- Numpy

## Reference
- Single Image Haze Removal Using Dark Channel Prior, Kaiming He, Jian Sun, and Xiaoou Tang, in CVPR 2009
